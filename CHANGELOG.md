# 更新日志

## [v1.1.1] - 2024-01-XX (Bug修复)

### 🐛 Bug修复
- **修复 `lsb_release: command not found` 错误**
  - 问题: Docker容器中默认未安装lsb-release包
  - 解决: 调整执行顺序,先安装基础工具再获取版本信息
  - 影响文件: 
    - `download-remote.sh`
    - `download-packages.sh`
    - `download-simple.sh`

### 🔧 技术改进
- 优化脚本初始化流程:
  1. 先执行 `apt-get update`
  2. 安装 `lsb-release`, `gnupg`, `wget` 等基础工具
  3. 然后再获取Ubuntu版本信息
- 调整步骤编号,使其更清晰

### 📝 文档更新
- 新增 `BUGFIX.md`: 详细的Bug修复说明文档

## [v1.1] - 2024-01-XX

### ✨ 新增功能
- **集成清华大学镜像源**: 大幅提升国内用户下载速度
  - 自动检测Ubuntu版本并配置对应的清华源
  - 支持DEB822格式(Ubuntu 24.04+)和传统格式
  - 优先使用清华PostgreSQL镜像
  - 自动备份原始源配置

### 🚀 优化改进
- 智能源选择机制:
  - 检测清华PostgreSQL源可用性
  - 不可用时自动回退到官方源
- 更详细的进度显示
- 增强的错误处理

### 📝 文档更新
- 新增 `清华源配置说明.md`: 详细的清华源配置指南
- 更新 `README.md`: 添加清华源特性说明
- 更新 `QUICKSTART.md`: 添加清华源特性说明
- 更新 `项目说明.md`: 添加清华源工作原理图解
- 更新所有脚本文件: 集成清华源配置逻辑

### 🔧 技术改进
- 改进Ubuntu版本检测逻辑
- 优化配置文件生成(适配不同Ubuntu版本)
