# 清华大学镜像源配置说明

## 📋 概述

本工具已集成**清华大学开源软件镜像站**的Ubuntu镜像,可大幅提升国内用户的下载速度。

## 🌟 主要特性

### 1. 自动检测与配置

脚本会自动:

- ✅ 检测Ubuntu版本代号 (如: noble, jammy, focal)
- ✅ 识别配置文件格式 (DEB822或传统格式)
- ✅ 备份原始配置
- ✅ 替换为清华源
- ✅ 优先使用清华PostgreSQL镜像

### 2. 智能源选择

```
检测系统版本
    ↓
判断配置格式
    ↓
Ubuntu 24.04+ ? → DEB822格式
Ubuntu <24.04 ? → 传统格式
    ↓
替换为清华源
    ↓
检查PostgreSQL清华源可用性
    ↓
可用 → 使用清华PostgreSQL源
不可用 → 使用官方PostgreSQL源
```

## 🔧 配置详情

### Ubuntu主仓库 (清华源)

**传统格式** (`/etc/apt/sources.list`):

```bash
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ noble main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ noble-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ noble-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ noble-security main restricted universe multiverse
```

**DEB822格式** (`/etc/apt/sources.list.d/ubuntu.sources`):

```yaml
Types: deb
URIs: https://mirrors.tuna.tsinghua.edu.cn/ubuntu
Suites: noble noble-updates noble-backports
Components: main restricted universe multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
```

### PostgreSQL仓库

**优先使用清华源**:

```bash
# 检测清华PostgreSQL源是否可用
if wget --spider -q "https://mirrors.aliyun.com/postgresql/repos/apt/dists/${CODENAME}-pgdg/"; then
    # 使用清华源
    deb https://mirrors.aliyun.com/postgresql/repos/apt noble-pgdg main
else
    # 回退到官方源
    deb https://mirrors.aliyun.com/postgresql/repos/apt noble-pgdg main
fi
```

## 🛠️ 配置示例

### 完整配置流程

```bash
# 1. 获取Ubuntu版本信息
lsb_release -cs
# 输出: noble (示例)

# 2. 备份原配置
cp /etc/apt/sources.list /etc/apt/sources.list.bak
cp /etc/apt/sources.list.d/ubuntu.sources /etc/apt/sources.list.d/ubuntu.sources.bak

# 3. 根据版本生成配置
UBUNTU_CODENAME=$(lsb_release -cs)

# 4. 写入清华源配置
cat > /etc/apt/sources.list << EOF
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ ${UBUNTU_CODENAME} main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ ${UBUNTU_CODENAME}-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ ${UBUNTU_CODENAME}-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ ${UBUNTU_CODENAME}-security main restricted universe multiverse
EOF

# 5. 更新包列表
apt-get update
```

### 启用源码仓库

如需下载源码包,取消配置中的注释:

```bash

```

因镜像站同步有延迟,可能需要使用官方安全更新源:

```bash

# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ noble-security ...

```

## 📝 故障排除

### 问题1: 清华源无法访问

**症状**:

```
Err: https://mirrors.tuna.tsinghua.edu.cn/ubuntu ...
Connection timed out
```

**解决**:

```bash
# 临时切换回官方源
mv /etc/apt/sources.list.bak /etc/apt/sources.list
apt-get update

# 或使用其他镜像
sed -i 's/tuna.tsinghua.edu.cn/mirrors.aliyun.com/g' /etc/apt/sources.list
```

### 问题2: GPG签名错误

**症状**:

```
The following signatures couldn't be verified
```

**解决**:

```bash
# 重新导入密钥
apt-key adv --keyserver keyserver.ubuntu.com --recv-keys ACCC4CF8
apt-get update
```

### 问题3: 某些包找不到

**原因**: 清华源可能缺少某些特殊仓库

**解决**:

```bash
# 添加额外仓库
add-apt-repository universe
add-apt-repository multiverse
apt-get update
```
